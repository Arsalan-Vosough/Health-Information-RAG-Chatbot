# Health Information RAG Chatbot

A multilingual (Finnish/English) retrieval-augmented generation (RAG) chatbot built with Flask, FAISS, and GPT. It answers health-related questions based on curated local documents (PDFs and JSON chunks).

## How It Works

1. **`embed.py`** — Reads PDFs and JSON chunks from the `sources/` folder, generates embeddings using a local HuggingFace model, and saves a FAISS vectorstore to disk.
2. **`app.py`** — Loads the saved FAISS index and runs a Flask web server. User questions are matched against the vectorstore and answered by GPT via LangChain.

## Setup

### 1. Clone the repository

```bash
git clone https://github.com/Arsalan-Vosough/Health-Information-RAG-Chatbot.git
cd Health-Information-RAG-Chatbot
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Set your OpenAI API key

Copy `.env.example` to `.env` and add your key:

```bash
cp .env.example .env
```

Then edit `.env`:

```
OPENAI_API_KEY=your-openai-api-key-here
```

Load it before running the app:

```bash
# Linux/macOS
export $(cat .env | xargs)

# Windows (PowerShell)
$env:OPENAI_API_KEY="your-openai-api-key-here"
```

### 4. Build the FAISS index

Run this once to embed your documents:

```bash
python embed.py
```

This will create the `sources/faiss_index/` folder.

### 5. Start the app

```bash
python app.py
```

Then open [http://localhost:5000](http://localhost:5000) in your browser.
